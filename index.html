Programmi di Kali Linux

Durante il corso, abbiamo utilizzato diversi programmi di Kali Linux per svolgere attività di sicurezza. Di seguito è una lista di alcuni di questi programmi e una breve spiegazione su cosa fanno:

  • Burpsuite: Tool di test della sicurezza delle applicazioni web.
  • Commix: Strumento per il rilevamento e l'esecuzione di attacchi di injection.
  • Skipfish: Scanner di sicurezza delle applicazioni web.
  • Kismet: Strumento di rilevamento e analisi delle reti wireless.
  • Wifite: Tool per il cracking delle password delle reti wireless.
  • Amass: Tool per l'analisi e l'enumerazione delle infrastrutture Internet.
  • Social Engineering Toolkit: Framework per condurre attacchi di ingegneria sociale.
  • Netsniff-ng: Suite di strumenti per il monitoraggio e l'analisi del traffico di rete.
  • Patator: Framework per il test della sicurezza e la penetrazione dei sistemi.
  • Nmap: Strumento di scansione delle reti e rilevamento dei servizi.
  • Exe2Hex: Strumento per convertire file eseguibili in formato esadecimale.
  • Chkrootkit: Strumento per il rilevamento di rootkit nel sistema.
  • Rkhunter: Strumento per il rilevamento di rootkit, backdoor e exploit nel sistema.
  • Cewl: Strumento per l'estrazione di parole chiave da pagine web e file di testo.
  • Hashcat: Strumento per il cracking delle password mediante attacchi di forza bruta.
  • Bulk Extractor: Strumento per l'estrazione di informazioni da immagini di disco e file system.
  • Metasploit: Framework per lo sviluppo e l'esecuzione di exploit.
  • Wireshark: Analizzatore di pacchetti di rete.
  • Armitage: Interfaccia grafica per il framework Metasploit.
  • Nikto: Scanner di sicurezza per il rilevamento di vulnerabilità delle applicazioni web.
  • Raever: Strumento per l'analisi del traffico di rete e l'intercettazione di dati.
  • Security Headers: Strumento per il controllo delle intestazioni di sicurezza HTTP.
  • SpiderFoot: Framework per l'automazione della raccolta di informazioni di intelligence.
  • Hydra: Strumento per il cracking delle password mediante attacchi di forza bruta.

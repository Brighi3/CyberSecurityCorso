def get_cvss_severity(score):
    if score < 4.0:
        return 'Low'
    elif score < 7.0:
        return 'Medium'
    elif score < 9.0:
        return 'High'
    else:
        return 'Critical'

# Funzione per eseguire la scansione delle porte
def run_scan(url, ports):
    vulnerabilities = []
    for port in ports:
        try:
            s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
            s.settimeout(2)
            result = s.connect_ex((url, port))
            s.close()

            if result == 0:
                vulnerabilities.append((port, 'Open'))
            else:
                vulnerabilities.append((port, 'Closed'))

        except socket.error:
            vulnerabilities.append((port, 'Error'))

    return vulnerabilities

Spiegazione del codice

Questo codice include una serie di funzioni per eseguire una scansione di vulnerabilità su un sito web:

  • CVSS_SEVERITY_MAP: Questa mappa associa i livelli di gravità CVSS ai rispettivi punteggi.
  • get_cvss_severity(score): Questa funzione restituisce la gravità CVSS corrispondente in base al punteggio fornito.
  • run_scan(url, ports): Questa funzione esegue la scansione delle porte specificate per il sito web fornito e restituisce una lista di vulnerabilità.
  • analyze_services(url): Questa funzione si connette al sito web specificato e restituisce i servizi associati al sito.
  • truncate_url(url): Questa funzione tronca l'header dall'URL fornito.
  • print_vulnerabilities(vulnerabilities): Questa funzione stampa i risultati della scansione delle vulnerabilità e le relative statistiche.
  • print_services(services): Questa funzione stampa i servizi associati al sito web.
  • save_results_to_file(file_name, vulnerabilities, vulnerabilities_by_type, vulnerabilities_by_severity, services=None): Questa funzione salva i risultati della scansione su un file.
  • main(): Questa è la funzione principale che gestisce il flusso dell'applicazione, richiedendo l'URL del sito web da analizzare, l'intervallo di porte e consentendo di salvare i risultati.

Il codice fornisce un modo per eseguire la scansione delle vulnerabilità di un sito web, stampare i risultati e salvarli su un file.
